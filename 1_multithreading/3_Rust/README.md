# Задача 3: Многопоточность в Rust: Генератор Мандельброта

## Описание
Данный проект реализует параллельную генерацию изображения множества Мандельброта.
Используется пул потоков (worker threads) для распределения вычислительной нагрузки. Изображение разбивается на строки, которые динамически распределяются между потоками через канал (mpsc).

## Структура
- `mandelbrot/` - директория с проектом Cargo.
  - `src/main.rs` - исходный код.
  - `input.txt` - конфигурация (размеры, потоки, координаты).
  - `Cargo.toml` - файл зависимостей.

## Сборка и запуск

1. Перейдите в директорию проекта:
   ```bash
   cd mandelbrot
   ```

2. Запустите проект через Cargo:
   ```bash
   cargo run --release
   ```
   *Флаг `--release` важен для производительности, так как вычисления интенсивны.*

3. Результат будет сохранен в файл, указанный в `input.txt` (по умолчанию `mandelbrot.ppm`).

## Конфигурация (input.txt)
Пример файла `input.txt`:
```
width: 800
height: 600
max_iterations: 1000
threads: 8
xmin: -2.0
xmax: 1.0
ymin: -1.0
ymax: 1.0
output_file: mandelbrot.ppm
```

## Просмотр результата
Файл `.ppm` можно открыть многими просмотрщиками изображений (например, GIMP, IrfanView, или онлайн конвертерами). В macOS можно использовать `Preview`.
