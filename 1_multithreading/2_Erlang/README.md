# Задача 2: Параллельное умножение матриц на Erlang

## Описание
Эта программа умножает две матрицы, считанные из файла `input.txt`, используя параллельные процессы в Erlang. Каждая строка результирующей матрицы вычисляется в отдельном процессе.

## Формат ввода (`input.txt`)
Входной файл должен содержать целые числа, разделенные пробелами:
1. Количество строк матрицы A
2. Количество столбцов матрицы A
3. Элементы матрицы A (построчно)
4. Количество строк матрицы B
5. Количество столбцов матрицы B
6. Элементы матрицы B (построчно)

Пример:
```
2 3
1 2 3
4 5 6
3 2
7 8
9 1
2 3
```

## Формат вывода (`output.txt`)
Выходной файл содержит результирующую матрицу, где элементы разделены пробелами, а строки находятся на новых линиях.

## Как запустить

### Требования
- Установленный Erlang/OTP.

### Шаги
1. Перейдите в директорию:
   ```bash
   cd 1_multithreading/Erlang
   ```
2. Скомпилируйте код:
   ```bash
   erlc matrix_mult.erl
   ```
3. Запустите программу:
   ```bash
   erl -noshell -s matrix_mult main
   ```
4. Проверьте результат:
   ```bash
   cat output.txt
   ```

## Дополнительные примеры для тестирования

### Пример 1: Умножение на единичную матрицу (3x3)
Этот пример проверяет корректность умножения квадратных матриц. Умножение любой матрицы на единичную должно дать исходную матрицу.

**Входные данные (`input.txt`):**
```
3 3
10 20 30
40 50 60
70 80 90
3 3
1 0 0
0 1 0
0 0 1
```

**Ожидаемый результат (`output.txt`):**
```
10 20 30
40 50 60
70 80 90
```

### Пример 2: Прямоугольные матрицы (2x4 и 4x2)
Более сложный пример с неквадратными матрицами для проверки корректной работы с размерностями.

**Входные данные (`input.txt`):**
```
2 4
1 2 3 4
5 6 7 8
4 2
1 2
3 4
5 6
7 8
```

**Ожидаемый результат (`output.txt`):**
```
50 60
114 140
```

### Пример 3: Смешанные положительные и отрицательные числа (3x3)
Тест на корректность работы со знаковыми целыми числами и арифметикой.

**Входные данные (`input.txt`):**
```
3 3
1 -2 3
-4 5 -6
7 -8 9
3 3
-1 2 -3
4 -5 6
-7 8 -9
```

**Ожидаемый результат (`output.txt`):**
```
-30 36 -42
66 -81 96
-102 126 -150
```
