# Задача 11: TCP Сервер Шифра Цезаря на Julia

Клиент-серверное приложение на языке Julia. Сервер принимает текст и сдвиг, возвращая зашифрованное сообщение.

## Требования
- Julia (версия 1.0+)

## Структура
- `server.jl` - Сервер
- `client.jl` - Клиент
- `input.txt` - Данные (формат `СДВИГ|ТЕКСТ`)

## Запуск

1. Запустите сервер:
   ```bash
   julia server.jl
   ```
2. В другом терминале запустите клиент:
   ```bash
   julia client.jl
   ```

## Особенности реализации
- Используются встроенные `Sockets`.
- Сервер асинхронный (`@async`), может обрабатывать несколько клиентов параллельно.
- Функция шифрования корректно обрабатывает регистр и цикличность алфавита.
