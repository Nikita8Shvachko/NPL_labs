# Задача 10: TCP Эхо-сервер на D

Простой клиент-сервер на языке D. Сервер разворачивает строки и добавляет смайлик `((:`, если строка — палиндром.

## Требования
- Компилятор DMD (D Compiler)

## Структура
- `server.d` - Код сервера
- `client.d` - Код клиента
- `input.txt` - Входные данные для клиента

## Запуск

1. **Компиляция:**
   ```bash
   dmd server.d
   dmd client.d
   ```

2. **Запуск сервера (в первом терминале):**
   ```bash
   ./server
   ```

3. **Запуск клиента (во втором терминале):**
   ```bash
   ./client
   ```

4. **Результат:**
   Смотрите файл `output.txt`.
