# Задача 6: Ханойская башня (Standard ML)

Решение задачи "Ханойская башня" на языке Standard ML.

## Структура
- `hanoi.sml`: Исходный код программы.
- `input.txt`: Входной файл, содержащий количество дисков (целое число).
- `output.txt`: Выходной файл с последовательностью ходов.

## Требования
Для запуска необходим компилятор Standard ML.
Рекомендуется использовать **MLton** (оптимизирующий компилятор) или **SML/NJ**.

### Установка на macOS
```bash
# Установка MLton (рекомендуется)
brew install mlton

# ИЛИ установка SML/NJ (требует прав администратора при установке через brew cask)
brew install smlnj
```

## Запуск с использованием MLton
1. Перейдите в директорию с заданием:
   ```bash
   cd 2_recursion/Standard_ML
   ```
2. Скомпилируйте программу (флаг необходим для поддержки кириллицы):
   ```bash
   mlton -default-ann 'allowExtendedTextConsts true' hanoi.sml
   ```
3. Запустите полученный исполняемый файл:
   ```bash
   ./hanoi
   ```
4. Результат будет записан в файл `output.txt`.

## Запуск с использованием SML/NJ (интерпретатор)
Если вы используете SML/NJ:
```bash
sml hanoi.sml
```

## Пример
**input.txt:**
```
3
```

**output.txt:**
```
Переместить диск 1 со стержня Старт на стержень Финиш
Переместить диск 2 со стержня Старт на стержень Вспом
Переместить диск 1 со стержня Финиш на стержень Вспом
Переместить диск 3 со стержня Старт на стержень Финиш
Переместить диск 1 со стержня Вспом на стержень Старт
Переместить диск 2 со стержня Вспом на стержень Финиш
Переместить диск 1 со стержня Старт на стержень Финиш
```
