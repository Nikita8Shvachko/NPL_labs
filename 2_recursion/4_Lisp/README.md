# Задача 4: Решатель Лабиринтов (Common Lisp)

## Описание
Эта программа находит путь в лабиринте, используя рекурсивный алгоритм DFS (поиск в глубину), реализованный на Common Lisp.

## Требования
- Реализация Common Lisp. Рекомендуется **SBCL (Steel Bank Common Lisp)**.

### Установка на macOS
Вы можете установить SBCL с помощью Homebrew:
```bash
brew install sbcl
```

## Использование

1. **Входные данные**: Убедитесь, что файл `input.txt` находится в той же директории.
   - `#` : Стена
   - ` ` : Пустое пространство (проход)
   - `S` : Точка старта
   - `E` : Точка выхода

   Пример `input.txt`:
   ```text
   ##########
   #S#      #
   # # #### #
   # # #    #
   #   # #E #
   # ### ####
   #   #    #
   ##########
   ```

2. **Запуск скрипта**:
   ```bash
   sbcl --script maze.lisp
   ```
   Или с помощью `clisp`:
   ```bash
   clisp maze.lisp
   ```

3. **Результат**:
   - Решенный лабиринт с путем, отмеченным символами `*`, будет выведен в консоль.
   - Результат также сохраняется в файл `output.txt`.
