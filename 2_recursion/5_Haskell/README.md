# Задача 5: N Ферзей (Haskell)

## Описание
Программа решает классическую задачу о расстановке N ферзей на шахматной доске размером N x N так, чтобы они не атаковали друг друга.

## Файлы
- `n_queens.hs`: Исходный код программы.
- `input.txt`: Входной файл, содержащий число N (размер доски).
- `output.txt`: Выходной файл с найденными решениями (создается после запуска).
- `TASK.md`: Описание задания.

## Запуск
Для запуска программы требуется компилятор GHC.

1. Убедитесь, что в файле `input.txt` указано желаемое число N (например, 8).
2. Скомпилируйте и запустите программу:

```bash
# Компиляция
ghc n_queens.hs -o n_queens

# Запуск
./n_queens
```

Или запустите напрямую через `runhaskell`:

```bash
runhaskell n_queens.hs
```

3. Проверьте результат в файле `output.txt`.
