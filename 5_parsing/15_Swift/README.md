# Задача 15: Калькулятор расходов на Swift

Парсер, который извлекает цены из текста, конвертирует их в рубли и считает общую сумму.

## Требования
- Swift компилятор (`swiftc`)
- macOS (или Linux с установленным Swift)

## Запуск

1. Скомпилируйте и запустите:
   ```bash
   swift parser.swift
   ```
   *Или скомпилируйте в бинарник:*
   ```bash
   swiftc parser.swift -o parser
   ./parser
   ```

## Пример работы
Входной текст с ценами в USD, EUR и RUB конвертируется по курсу (USD=92.5, EUR=99.8) и суммируется.
