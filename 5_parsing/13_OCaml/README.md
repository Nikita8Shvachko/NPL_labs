# Задача 13: Парсер почтовых доменов на OCaml

Программа парсит email-адреса в тексте и подсчитывает статистику по доменам.

## Требования
- Компилятор OCaml (`ocamlc` или интерпретатор `ocaml`)

## Запуск

1. Скомпилируйте и запустите программу (используя библиотеку `str`):
   ```bash
   ocaml str.cma parser.ml
   ```
   *Примечание: `str.cma` - это библиотека регулярных выражений, которая не линкуется по умолчанию, поэтому её нужно указывать явно.*

2. Или скомпилируйте в бинарный файл:
   ```bash
   ocamlc str.cma -o parser parser.ml
   ./parser
   ```

## Результат
Программа создаст файл `output.txt` со статистикой:
```text
gmail.com: 3
mysite.org: 2
corp.io: 1
...
```
