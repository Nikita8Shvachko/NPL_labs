# Задача 14: Парсер дат на Scala

Программа ищет даты в тексте (поддерживает форматы `YYYY-MM-DD` и `DD-MM-YYYY`), нормализует их, удаляет дубликаты и сортирует хронологически.

## Требования
- Scala (версия 2.13 или 3.x)
- sbt (опционально, можно запускать просто через `scala`)

## Запуск

1. Запустите скрипт через консольную утилиту Scala:
   ```bash
   scala DateParser.scala
   ```

2. Или скомпилируйте и запустите:
   ```bash
   scalac DateParser.scala
   scala DateParser
   ```

## Пример
Вход: `2023-01-15` и `15-01-2023` распознаются как одна и та же дата.
Выходной файл будет содержать отсортированный список в формате ISO (`YYYY-MM-DD`).
