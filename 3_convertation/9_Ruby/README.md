# Задача 9: Конвертер Markdown в HTML на Ruby

Программа для конвертации текстовых файлов формата Markdown в HTML страницы.

## Описание

Скрипт читает файл `input.md`, парсит основные элементы разметки Markdown (заголовки, списки, жирный текст, курсив, ссылки) и генерирует полноценную HTML страницу в файле `output.html`.

## Требования

- Ruby (любая современная версия)

## Установка и запуск

1.  Убедитесь, что Ruby установлен:
    ```bash
    ruby -v
    ```
2.  Перейдите в директорию проекта:
    ```bash
    cd 3_convertation/9_Ruby
    ```
3.  Запустите скрипт:
    ```bash
    ruby converter.rb
    ```

## Поддерживаемый синтаксис

**Входной (Markdown):**
```markdown
# Тест возможностей Markdown конвертера

## 1. Заголовки разных уровней
### Заголовок 3 уровня
#### Заголовок 4 уровня
##### Заголовок 5 уровня
###### Заголовок 6 уровня

**жирный текст с двойными звездочками**  __жирный текст с подчеркиваниями__.
*курсив со звездочками*  _курсив с подчеркиваниями_.
 **жирный** и *курсив* 

## 3. Списки
- Первый элемент
- Второй элемент
- Третий элемент

Список с использованием звездочек:
* Яблоки
* Груши
* Апельсины

Вот ссылка на [Google](https://google.com).
```

**Выходной (HTML):**
```html
<h1>Заголовок</h1>
<p>Текст с <strong>жирным</strong> и <em>курсивом</em>.</p>
<ul>
  <li>Элемент списка</li>
</ul>
<p><a href="http://example.com">Ссылка</a></p>
```
