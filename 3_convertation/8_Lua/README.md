# Задача 8: Конвертер JSON в XML на Lua

Программа для конвертации списка сотрудников из формата JSON в XML.

## Описание

Скрипт читает файл `input.json`. Поскольку в стандартной библиотеке Lua нет парсера JSON, реализован простой парсер на основе pattern matching (регулярных выражений Lua), который извлекает объекты и их поля. Результат записывается в `output.xml` в требуемом формате.

## Требования

- Lua (версия 5.1 или выше)

## Установка и запуск

1.  Убедитесь, что Lua установлен:
    ```bash
    lua -v
    ```
2.  Перейдите в директорию проекта:
    ```bash
    cd 3_convertation/Lua
    ```
3.  Запустите скрипт:
    ```bash
    lua converter.lua
    ```
